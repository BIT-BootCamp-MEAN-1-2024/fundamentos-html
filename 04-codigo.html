HTML, Hypertext Markup Languaje

HTML, o Lenguaje de Marcado de Hipertexto, es la base para la creación de páginas web. Es un lenguaje de marcado, lo que significa que utiliza etiquetas para definir la estructura y el contenido de una página web. Estas etiquetas son instrucciones para los navegadores web sobre cómo mostrar el contenido, como texto, imágenes y videos.

Si bien HTML por sí mismo no agrega estilo visual, proporciona el marco para las páginas web y les permite ser interactivas. Es uno de los lenguajes fundamentales de la World Wide Web, junto con CSS y JavaScript.

Ejemplo:

            <!DOCTYPE html>
            <html>
            <head>
                <title>Ejemplo de código</title>
            </head>
            <body>
                <h1>Hola mundo!</h1>
            </body>
            </html>

Este código HTML simple crea una página web con un título ("Mi primera página web") y un párrafo que dice "Este es mi primer párrafo".
